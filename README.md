# StockTradeSimulator - 股票模拟交易平台

[中文](#介绍) | [English](#Introduction)

## 介绍 🚀
一个基于Web的实时股票模拟交易系统，提供虚拟股市交易体验。包含实时行情、持仓管理、成就系统、收益排名等功能，帮助用户学习股票交易机制，无真实金融风险。

![项目截图](screenshot.png)

## 主要功能 💼
- 📈 **实时模拟股市** - 21支虚拟股票10秒刷新价格
- 💰 **交易系统** - 支持买入/卖出、全仓/半仓操作
- 📊 **持仓管理** - 实时计算持仓收益与总资产
- 🏆 **成就系统** - 10个交易成就等你解锁
- 🏅 **收益排名** - 多维度股票涨跌幅排行
- 📱 **响应式设计** - 完美适配PC/手机端
- 💾 **本地存储** - 自动保存交易进度
- ⚠️ **风险提示** - 显眼免责声明确保合规

## 技术栈 🔧
- 前端三件套：HTML5 / CSS3 / JavaScript
- 本地存储：LocalStorage
- 数据模拟：随机波动算法
- 界面：CSS变量实现明暗主题
